Description of the Course

Data structures are a way of organizing and storing data so that they can be accessed and worked with efficiently. They define the relationship between the data, and the operations that can be performed on the data. There are many different types of data structures, each with its own advantages and disadvantages. Some data structures are highly specialized, while others (like arrays) are more generally used.

Topics Covered in This Course

  • Arrays
  • Linked Lists
  • Stacks
  • Queues
  • Trees
  • Graphs
  • Hash Tables
